== Description ==
This Plugin is currently in development and in a very early and unfinished state. Please check back later.
If you like to contribute to the project, feel free to send me a message.

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/super-easy-picklist-for-woocommerce directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‚Plugins‘ screen in WordPress.
3. Enjoy
